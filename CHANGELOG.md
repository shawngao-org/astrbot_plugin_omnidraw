# Changelog

## 3.3.2

- 修复配置页临时预览 URL 被保存进多组人设配置，导致刷新后显示“1 图”但预览空白的问题。
- 保存配置时会把当前有效的预览 token 反解为真实本地图片路径，并自动跳过历史遗留的失效预览 token。

## 3.3.1

- 修复多组人设中较大的本地参考图刷新后只显示数量、不显示预览的问题。
- 大图预览改为通过受控 token API 回传，避免把本地路径直接交给浏览器。

## 3.3.0

- 新增多组人设档案，支持每个人设独立维护 Base Prompt 与参考图组。
- 新增 `/人设` 与 `/切换人设 [序号/ID/名称]`，可在指令侧热切换自拍人格。
- 配置页新增人设组管理，切换人设时同步切换对应参考图矩阵。
- 保持旧版单人设字段兼容，旧配置会自动迁移为第一组人设。

## 3.2.1

- 深度优化提示词副脑，强制成功优化结果使用英文输出。
- 精进内置风格矩阵，尤其强化“自拍专用极致真实”的日常手机自拍质感。
- 增加人体结构、自拍角度、皮肤质感、反光控制、畸形规避等真实感护栏。

## 3.2.0

- 统一插件版本、作者和 metadata 注册信息。
- 按 AstrBot 数据目录规范持久化配置与参考图。
- 增强配置归一化，兼容旧模型逗号写法和多参考图。
- 修复副脑链路保存位置、空 API Key 崩溃、视频参数丢失和 URL 拼接问题。
- 增加后台视频任务生命周期清理。
- 修复配置页潜在 HTML 注入、预设冒号截断和桌面端卡片样式缺失。
